執行sliding_puzzle.m開始遊戲

sliding puzzle 滑塊拼圖
遊戲方式
1.選擇遊戲圖片：點擊圖片(點擊最上面標題顯示遊戲方式)

2.選擇遊戲難度

3.點擊任意鍵開始遊戲

4.遊戲可由兩種方式進行

(1)滑鼠點擊圖片方塊

(2)按方向鍵

5.按R顯示遊戲方式、按S顯示目標圖片

6.按「enter」退出此次遊戲


目標

移動圖片方塊還原原圖

結果

顯示花費的步數、時間


程式碼

顯示遊戲方式
製作並顯示遊戲封面
判斷使用者選取的圖片
讓使用者選擇難易度
按照使用者選擇的難度進行圖片隨機打亂，越難方塊數量越多
判斷使用者執行的方式，並移動圖片方塊
當玩家成功還原圖片時結束這次遊戲，顯示花費的步數及時間，並讓玩家選擇下一次遊戲或退出
